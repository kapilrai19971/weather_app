Index | Git Commands:

1. git init

To initialize the git to project,

For example: $git init

2. git status

To check the untracked file status before commiting or adding to git.

For example: $ git status

3. git add filename

To add single file to git cache

For example: $ git add index.html

4. git add .

To add all files to git cache if changes is done

For example: $ git add .

5. git rm --cached filename

To remove the file from git cache

For example: $ git rm --cached index.html

6. git restore filename

To remove all new changes are not added to git cache

For example: $ git restore index.html

7. git commit -m "message"

To add to git file. Here -m is used to add message for commit

For example: $ git commit -m "Second commit "

8. git remote add origin https|ssh repo link

To add the remote server repository to local server

For example: $ git remote add origin git@github.com:kapilrai19971/weather_app.git

9. git show

To check the file content

For example: $ git show

10. git push -u origin branchname

To push our changes to remote server

Note: If we want to push code or contents in master branch then branchname or other branch name

For example: $ git push -u origin master
